## Bachelors Thesis Project

## Overview
This project is my BSc thesis analyzing **ΛCDM cosmology parameters** under the assumption of a **flat Universe** (Ωₖ = 0).  
It includes code to reproduce the evolution of **matter, dark energy, and radiation densities** as a function of redshift.

## Data
The analysis uses **supernova distance modulus data** from the Union compilation of the **Supernova Cosmology Project (SCP, 2008)**.  

**Reference:**  
Kowalski, M., et al., 2008. *Astrophysical Journal*, 686: 749–778. [doi:10.1086/589937](https://doi.org/10.1086/589937)
